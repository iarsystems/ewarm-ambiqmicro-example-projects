Blinky Example

DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for ARM to
  develop code for Apollo1-EVB board.
  This example consists of a simple test that will blink various patterns on the
  5 LEDS of the Apollo1-EVB (evaluation board, March 2017).

  Note - this example will not work on the older Apollo EVK board without
  modification as the 4 LEDs of that board are on different GPIO pins.

CONFIGURATION
=============
  The Workspace contains Blinky example with the configuration:

  Flash Debug: The example is flashed into the embedded flash memory of APOLLO512-KBR
  microcontroller

COMPATIBILITY
=============
  The example project is compatible with Apollo1-EVB board.
  By default the project is configured to use I-Jet debugger.

GETTING STARTED
===============
  1) Start the IAR Embedded Workbench for ARM.

  2) Connect the debugger to the Apollo1-EVB board.

  3) Power on the board.

  4) Press Ctrl+D or use Download and Debug button to start a debug session.
